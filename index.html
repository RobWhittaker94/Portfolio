Wordle Clone



Overview:

A game where you have 6 attempts to work out a randomly generated word. The tiles containing each letter from your guess flip and change to a colour; orange to signify if the letter exists in the word but in the wrong position, green if it is correct and in the right position. I've left the swear words in so you might get one if you're lucky!


Details:

JavaScript based with an API call for the random word generator which I then proggrammed to only give 5 letter long words. As the game already exists it was a great opportunity to learn how to convert game logic into code without having to think up the game itself. I used CSS to produce an efficient and clean UI with a pleasing aesthetic.

"ShouldIWearMyShortsToday?"



Overview:

A fun React app inspired by a simple geolocation project I was building following a course. The app is to help you decide whether the weather is suitable for you to wear your shorts today as living in the UK I found that sometimes looking out the window wasn't enough. I added a loading screen with a prompt for the user to accept the location request and a placeholder link to amazon.com/shorts until I get a shorts retailer partnership.


Details:

Built with React. The app takes your location and cross-checks the time of year with which hemisphere you are living in to produce a results screen. I am currently working on adding various API calls such as an in depth weather/temperature forecast, amount of sun time left and more precise geolocation logic so that people at the north pole won't be advised to wear shorts even if it is the summer.

Dice Game



Overview:

In this game 2 players race to score 100 points by rolling a dice. The value of each roll is added to the last. Each player takes a gamble on whether they should 'bank' (hold) their current score, ending their turn or continue rolling to see how high they can go. If the dice rolls a 1 then your streak ends and whatever you haven't 'banked' is lost and the dice is passed over to the other player, highlighted by the section changing colour.


Details:

The Dice game is a simple DOM manipulation project that utilises game mechanices and user state to change the styling and HTML of the project. It was initially built as a project with a JavaScript course but I enjoyed developing it so much that I decided to carry on with it.

Matrix Landing Page



Overview:

After entering the world of coding I couldn't resist giving this a go. I followed a youtube tutorial on how to recreate the famous digital rain and then took it forward as a fun idea for a portfolio landing page background. Including an animated custom terminal display to produce a series of messages. I updated the character set and adjusted the speed and fade-out timers for the rain to make it look more like the imagery from the film.


Details:

JavaScript based. I used this project as an opportunity to focus on making my code as efficient and easy to read as possible, avoiding the boilerplate approach, lengthy index.js files and hard-coded style application by linking files to eachother inspired by how React projects are organised.
